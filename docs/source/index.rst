Installation
------------

To install anormalratingsystem, simply run:

.. code-block:: bash

    pip install anormalratingsystem

Usage
-----

After installation, you can use anormalratingsystem by
running:

.. code:: bash

    $ python3 -m anormalratingsystem
    # or
    $ anormalratingsystem
